# OpportunityLab Development Roadmap

This roadmap is intended as a guide rather than a strict feature list.
Packages may change as development progresses.

---

Version 0.19
Codename: Prospector

Theme:
"Every opportunity begins with discovery."

Primary Goals

• Stabilise the AI architecture.
• Introduce AI Controller.
• Improve Provider Registry.
• Improve AIRouter.
• Related Intelligence.
• AI cache.
• Thread safety.
• Testing framework.
• Developer utilities.
• Cleaner application architecture.

Focus

Build strong foundations before expanding features.
